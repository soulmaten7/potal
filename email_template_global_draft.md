## Usage Guide / 사용 가이드:

### Key differences from US Domestic template / US Domestic 템플릿과의 차이점:
- Focus on "access to US consumers" instead of "equal visibility vs Amazon"
  "Amazon 대비 공정 노출" 대신 "미국 소비자 접근"에 초점
- Highlight Total Landed Cost transparency as the key value
  총비용(TLC) 투명성을 핵심 가치로 강조
- Emphasize removing cross-border purchase hesitation
  해외구매 망설임 제거를 강조
- Global expansion = early partners already integrated for new markets
  글로벌 확장 = 초기 파트너는 새 시장에서 이미 연동 완료
- API request includes shipping rates (needed for TLC calculation)
  API 요청에 배송비 포함 (TLC 계산에 필요)
