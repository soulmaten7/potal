Show real-time Total Landed Cost on WooCommerce product pages. 240 countries, duties, taxes, customs fees — all in one widget.

== Description ==

POTAL calculates and displays the Total Landed Cost for international orders, including:

* Import duties based on HS code classification
* VAT/GST for 240 countries
* Customs processing fees
* De minimis threshold detection
* DDP checkout (optional) — buyer pays once, no surprise fees

= Features =

* **Auto HS Code Classification** — AI classifies products into HS codes automatically
* **240 Countries** — Full global coverage with real duty rates
* **DDP Checkout** — Add import duties to cart total (Delivered Duty Paid)
* **Multi-currency** — Display prices in buyer's local currency
* **Shortcode Support** — Place the widget anywhere with [potal_landed_cost]

== Installation ==

1. Upload the plugin to `/wp-content/plugins/potal-landed-cost/`
2. Activate through the 'Plugins' menu
3. Go to WooCommerce → POTAL Landed Cost
4. Enter your API key from [potal.app/dashboard](https://www.potal.app/dashboard)
5. Set your origin country and widget position
