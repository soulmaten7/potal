## 분석 내용

### 1. 패턴 E 106건 전체 추출

v6 에러 분석 결과에서 패턴 E로 분류된 106건 전체. 각 건에 대해:
```
- product_name 원문
- product_attributes 전체
- seller_category
- v6가 답한 Chapter + WCO 설명
- 정답 Chapter + WCO 설명
- product_name에 있는 정답 단서 키워드 (어떤 단어가 정답을 알려주는지)
```

### 2. 못 잡은 단서 분류

106건의 "정답 단서 키워드"를 패턴별로 분류:

```
E-1: "전문 용어" — 일반인은 모르는 전문 키워드
  예: "carbon brush" = 전기 부품(Ch.85)인데 LLM이 brush=도구(Ch.82)로 판단
  예: "bushing" = 기계 부품(Ch.84)인데 LLM이 일반 금속(Ch.73)으로 판단
  → 키워드 자체는 있지만, 전문 지식 없이는 올바른 Chapter를 모름

E-2: "다의어" — 같은 단어가 여러 Chapter에 해당
  예: "brush" = 칫솔(Ch.96)? 페인트붓(Ch.96)? 전기 카본브러시(Ch.85)? 헤어브러시(Ch.96)?
  예: "ring" = 반지(Ch.71)? O-링(Ch.40)? 커튼링(Ch.83)?
  → 키워드는 있지만 문맥 없이는 Chapter 구분 불가

E-3: "복합 상품" — 여러 Chapter에 걸친 상품
  예: "LED light with plastic cover" = LED(Ch.85)? 플라스틱(Ch.39)? 조명(Ch.94)?
  → 주요 기능/소재 판단이 필요한데 LLM이 잘못 판단

E-4: "WCO 특수 규칙" — WCO Notes에 의해 특정 Chapter로 분류되는 경우
  예: 완구인데 전자 부품 포함 → Ch.95(완구)가 맞는데 LLM이 Ch.85(전자)
  예: 의류 세트인데 니트+우븐 혼합 → Ch.61? Ch.62? Notes 규칙으로 결정
  → WCO Section/Chapter Notes 규칙을 모르면 못 맞추는 경우

E-5: "키워드 위치" — 정답 단서가 product_name에 있지만 너무 뒤에/흐려서 LLM이 놓침
  예: "2024 New Fashion Hot Sale Vintage Style Beautiful Handmade Crystal Pendant Necklace"
  → "pendant necklace"가 맨 뒤에 있고 앞에 마케팅 문구가 길어서 LLM이 핵심을 못 잡음

E-6: "기타"
```

### 3. 각 하위 패턴별 건수 + 대표 사례

```
| 하위 패턴 | 건수 | 비율(106건 중) | 대표 키워드 | 해결 가능성 |
```

각 하위 패턴에서 자주 등장하는 키워드 목록 추출.

### 4. 정답 Chapter별 분포

```
| 정답 Chapter | 건수 | 설명 | 주요 하위 패턴 |
예: Ch.71(보석) 가 E패턴에서 가장 많은지? Ch.85(전기)? Ch.39(플라스틱)?
```

### 5. 하위 패턴 간 관계 분석

```
- E-1과 E-2가 겹치는 건이 있나? (전문 용어이면서 다의어)
- E-3과 E-4가 겹치는 건이 있나? (복합 상품이면서 WCO 특수 규칙)
- 가장 근본적인 원인이 뭔지 파악 (LLM 능력 한계? 정보 구조 문제? WCO 체계 복잡성?)
```

⚠️ 해결책은 찾지 않는다. 분류만 정확히 한다.
