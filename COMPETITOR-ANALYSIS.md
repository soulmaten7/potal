## 2. 경쟁사 스펙 비교

### Zonos (미국, 시장 1위급)
- **지원 국가:** 235개국
- **HS Code:** 10자리까지 지원, AI 자동분류 (연 10,000건 무료)
- **가격:** 셋업 $2,000~$4,000 + 주문당 $2 + 관세의 10%
- **특징:** Shopify/BigCommerce 네이티브 통합, 보장(guaranteed) 계산
- **대상:** 대형 이커머스 셀러

### Avalara AvaTax Cross-Border (미국, 엔터프라이즈급)
- **지원 국가:** 180개국+
- **HS Code:** AI 기반 자동분류, 국가별 세분화
- **가격:** 연간 ~$18,000+ (커스텀 견적)
- **특징:** ERP 통합, 세금 컴플라이언스 포함, 엔터프라이즈 SLA
- **대상:** 중대형 기업, 글로벌 물류

### SimplyDuty (영국)
- **지원 국가:** 110개국
- **HS Code:** 10자리, API + 온라인 계산기
- **가격:** £9.99~£499.99/월 (100~5,000 API 콜)
- **특징:** 간단한 REST API, 낮은 진입장벽
- **대상:** 소규모 셀러, 개인

### Dutify (신생)
- **지원 국가:** 100개국+
- **HS Code:** 국가별 풀 코드, MFN 세율 포함
- **가격:** 무료 플랜 포함, 세부 미공개
- **특징:** 이커머스 체크아웃 직접 통합, 위험물 경고
- **대상:** 이커머스 스타트업

## 4. POTAL의 현재 위치 분석

### 🟢 강점 (Phase 5로 크게 성장한 부분)
1. **가격 경쟁력 최강** — Free 티어 5,000 콜/월, Growth $49/월. Zonos 셋업비($2K)만으로 POTAL 3년 사용 가능
2. **FTA 처리 독보적** — 27개 FTA 협정 내장은 SimplyDuty보다 우위. 원산지→목적지 자동 FTA 매칭
3. **개발자 친화적** — 간단한 REST API, 빠른 온보딩, 셋업비 없음
4. **성장 속도** — 한 번의 Phase에서 HS Code +556%, 국가 +163%, FTA +125% 성장

### 🟡 현재 격차 (개선 필요)
1. **국가 커버리지 갭** — 21개국 vs Zonos 235개국 (약 9%)
2. **HS Code 깊이** — 6자리 vs 경쟁사 10자리 (6자리가 대부분 충분하지만 정밀도 차이)
3. **HS 엔트리 수** — 407개 vs 경쟁사 수만~수십만 (키워드 매칭 한계)
4. **AI 분류 없음** — 키워드 기반 vs 경쟁사 AI/ML 기반 자동분류
5. **Shopify 앱 미완** — Phase 3에서 해결 예정

### 🔴 전략적 격차 (장기 과제)
1. **Guaranteed 계산** — Zonos는 관세 정확도를 보장, POTAL은 추정치
2. **실시간 관세율 업데이트** — 경쟁사는 DB 자동 업데이트, POTAL은 수동
3. **엔터프라이즈 기능** — ERP 통합, 감사 로그, 컴플라이언스 리포팅

## 5. 시장 포지셔닝

```
가격 (높음)
  │
  │  Avalara ($18K+/yr)
  │     ● 엔터프라이즈
  │
  │        Zonos ($2K+ setup)
  │           ● 대형 셀러
  │
  │  SimplyDuty (£10~500/mo)
  │     ● 중소형
  │
  │     Dutify (미공개)        ★ POTAL ($0~99/mo)
  │        ● 스타트업              ● 개발자/SMB
  │
  └──────────────────────────────── 기능 (많음)
```

**POTAL의 Sweet Spot:** 가격 대비 기능이 가장 뛰어난 포지션. 특히 **개발자 + 중소 이커머스 셀러** 시장에서 SimplyDuty/Dutify와 직접 경쟁하되, 가격과 FTA 처리에서 우위.
