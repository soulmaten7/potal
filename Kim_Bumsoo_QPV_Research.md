## 3. 투자 철학 & 성향 — 피치덱 맞춤 전략의 핵심

### 3가지 핵심 투자 기준

#### ① 창업가의 행동력 (Founder's Action/Execution)
> "말만 하는 사람이 아니라, 실제로 움직이는 사람"

- 아이디어만 있는 사람 ❌ → **이미 뭔가를 만든 사람** ✅
- MVP가 있고, 이미 실행하고 있는 창업자를 선호
- **POTAL에 적용**: MVP 완성 + 웹사이트 배포 + iOS 앱 제출 + 파트너십 이메일 발송 중 → 행동력 증명 가능

#### ② 지적 정직성 (Intellectual Honesty)
> "자기 사업의 약점을 솔직하게 인정하는 사람"

- 과장하거나 약점을 숨기는 창업자 ❌ → **문제를 직시하고 해결하려는 태도** ✅
- 현재 뭐가 안 되는지, 어떤 리스크가 있는지를 솔직하게 말하는 것
- **POTAL에 적용**: "트래픽 아직 없음", "API 의존도 높음", "1인 개발" 등을 솔직하게 인정하면서도 해결 방향을 제시

#### ③ 사람 (People Over Product)
> "아이템은 말만 되면 된다. 사람이 중요하다."

- 아이디어/제품보다 **창업자 자체**를 봄
- 이 사람이 피벗하더라도 성공할 수 있는 사람인지
- 끈기, 학습 속도, 적응력
- **POTAL에 적용**: 코딩 경험 없이 바이브코딩으로 MVP 완성 → 학습 속도/적응력 어필

### 추가 투자 성향

#### Anti-Hype (반-유행)
- 하이프(Hype)에 올라타지 않고 자신만의 기준을 바탕으로 **가치투자**
- 기준에 합당하지 않은 하이프나 "남들이 하니까" 하는 투자는 최대한 지양
- 독자적인 이유로 이 사업을 하는 것이 중요

#### 생성형 AI에 대한 확신
- 투자 시장을 이끌 기술 화두로 **단연 생성형 AI**를 꼽음
- "인공지능이 인류의 삶을 큰 스케일로 바꿀 거라는 생각은 확고하다"
- **→ POTAL이 바이브코딩(AI)으로 만들어졌다는 점이 큰 어필 포인트**

#### 창업가에 대한 적극적 팔로업 선호
- 며칠 후 적극적으로 팔로업하거나 추가 작업물을 들고 오는 창업가를 선호
- 아이디어 단계에서 벗어나지 못하는 창업가보다 **실행하는 창업가**를 훨씬 높이 평가

#### 극초기 (Pre-Seed/Seed) 전문
- 60%가 1년 미만 기업 → **POTAL 현재 단계에 완벽히 부합**
- 매출이 없어도 됨
- 사업이 "말이 되면" (makes sense) 투자

#### 글로벌 M&A Exit 선호
- IPO보다 **글로벌 대기업에 인수되는 Exit** 선호
- POTAL 관점: "Google Shopping, Amazon 등에 인수될 수 있는 기술/데이터" 포지셔닝 가능

## 5. POTAL 피치덱 전략 — 김범수 대표 맞춤

### 핵심 메시지 프레임워크

| 김범수 기준 | POTAL 어필 포인트 |
|------------|-----------------|
| 행동력 | MVP 완성, 웹+iOS 배포, 6개 쇼핑몰 연동, 7곳 파트너십 이메일 발송 — 코딩 경험 없이 바이브코딩으로 혼자 구축 |
| 지적 정직성 | 트래픽 아직 부족, API 의존도 높음, 1인 개발 한계 — 그러나 해결 방향 명확 |
| 사람 | 비개발자가 AI로 MVP 만들고 실행 중 — 학습 속도, 끈기, 실행력 |
| 극초기 | Pre-seed 단계, 매출 0, 그러나 제품 작동 중 |
| 글로벌 Exit | 가격비교 데이터+기술 → Google Shopping, Amazon 등에 인수 가능성 |
| Anti-Hype | AI 쇼핑 비교는 유행이 아닌 실제 소비자 pain point 해결 |

### 피치덱 구조 제안 (김범수 대표 맞춤)

1. **Problem** — 온라인 쇼핑에서 진짜 최저가를 아는 사람은 없다
2. **Solution** — POTAL: 실시간 가격 + 배송비 + 관세 = Total Landed Cost 비교
3. **Demo** — 실제 검색 결과 스크린샷 (Domestic vs Global)
4. **Traction** — MVP 완성, 웹+iOS, 6개 쇼핑몰, 파트너십 진행 중
5. **Market** — 글로벌 크로스보더 이커머스 $1.21T, 미국 온라인 쇼퍼 2.75억명
6. **Business Model** — 어필리에이트 커미션 + 향후 스폰서 리스팅/프리미엄
7. **Why Now** — 크로스보더 커머스 폭발 + 기존 비교 도구 배송비/관세 미반영
8. **Why Me** — 비개발자가 바이브코딩으로 혼자 MVP 완성, 실행력 증명 (→ 행동력)
9. **Honest Challenges** — 트래픽 부족, API 의존, 1인 한계 (→ 지적 정직성)
10. **Ask** — 투자금 용도: 미국 법인 + API 확장 + 마케팅

### 주의사항
- **과장 금지** — 김범수 대표는 과장을 싫어함. 있는 그대로.
- **숫자가 없으면 숫자 없다고** — "유저 아직 없습니다. 하지만 이렇게 하고 있습니다"
- **사업보다 사람** — "이 아이디어가 대단합니다"보다 "저는 이렇게 움직였습니다"
- **글로벌 Exit 가능성** — 한국 내수 사업이 아님을 강조. 처음부터 미국/글로벌.

## 7. 바이브코딩 트렌드 — POTAL의 핵심 어필 포인트

### 김범수 대표가 중시하는 것과의 연결
- 김범수 대표는 **생성형 AI가 인류의 삶을 큰 스케일로 바꿀 것**이라고 확신
- POTAL은 바이브코딩(AI 코딩)으로 만들어진 제품 → **AI 시대의 실증 사례**
- 비개발자가 AI로 실제 작동하는 MVP를 만들었다는 것 자체가 어필

### 2025년 바이브코딩 시장 현황 (피치덱 참고)
- Y Combinator 2025년 겨울 배치: **25%가 코드베이스의 95%를 AI로 작성**
- 바이브코딩 시장 규모: $4.7B (2024) → $37B (2032) 전망
- Base44 (1인 바이브코딩 스타트업): 설립 6개월 만에 Wix에 $80M (약 1,100억원)에 인수
- Cursor (바이브코딩 도구): $29.3B 가치 평가, $2.3B 투자 유치
- 핵심 메시지: **"코드를 잘 짜는 사람"이 아닌 "문제를 잘 정의하는 사람"이 성공하는 시대**
- POTAL에 적용: 은태님은 코딩 경험 없지만, **"소비자가 진짜 최저가를 모르는 문제"를 정확히 정의**하고 AI로 해결
