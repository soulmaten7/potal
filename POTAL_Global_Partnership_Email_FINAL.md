## Usage Guide:

### Customize per store:
- Replace **[Store Name]** with each store's actual name
- Attach the **Global** version of the PDF proposal
- Communication is email-only (no phone calls)
- Send to each store's partnership/business development/affiliate email

### Key differences from US Domestic template:
- Focus on "access to US consumers" instead of "equal visibility vs Amazon"
- Highlight Total Landed Cost transparency as the key value
- Emphasize removing cross-border purchase hesitation
- Global expansion = early partners already integrated for new markets
- API request includes shipping rates (needed for TLC calculation)
- Does NOT list unintegrated global store names (only US stores already in progress)
