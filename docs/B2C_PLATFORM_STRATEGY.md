## 1. 핵심 컨셉

**"AI한테 일자리 뺏기지 마라. 여기에 상품을 올려라."**

전 세계 누구나 어필리에이트 링크로 상품을 올리면, 모든 국가의 소비자와 AI 에이전트가 해당 상품을 발견하고 구매할 수 있는 플랫폼.

### POTAL과의 관계
- **POTAL (B2B)**: 관세 계산 인프라 API/MCP/위젯 — 그대로 유지
- **새 플랫폼 (B2C)**: 별도 이름, 별도 도메인, 별도 프로젝트
- **연결점**: 새 플랫폼의 Landed Cost 엔진 = POTAL B2B 엔진 활용
- **두 플랫폼은 완전히 다른 제품**이지만, 인프라를 공유

---

## 2. 플랫폼 구조

### 셀러(어필리에이터) 측
```
가입 → 개인 페이지 생성 → 상품 등록 (어필리에이트 링크 + 상품 정보) → 자기 페이지 공유 → 수익 발생
```

- 누구나 가입 가능 (개인, 개인셀러, 기업 무관)
- 어필리에이트 링크 = 셀러 것 그대로 (플랫폼이 수수료 안 뗌)
- 개인 페이지에서 본인 상품 관리
- 여러 국가의 어필리에이트 프로그램 동시 등록 가능

### 소비자/AI 측
```
국가 기반 자동 필터 → 해당 국가로 배송 가능한 상품만 표시 → Landed Cost 자동 계산 → 비교 → 구매 (어필리에이트 링크)
```

- 사용자 위치(국가) 기반으로 배송 가능 상품 자동 필터링
- POTAL 엔진으로 관세+세금+배송비 = Total Landed Cost 자동 계산
- AI 에이전트가 구조화된 데이터를 가져갈 수 있는 구조 (API/MCP)

### 어필리에이트 플랫폼 측
```
플랫폼이 직접 입점 요청 → 어필리에이트 가입 가이드 등록 → 셀러들이 해당 플랫폼 상품도 올림
```

- 국가별 어필리에이트 가입 가능한 사이트 목록 제공
- 가입 방법 가이드 제공
- 어필리에이트 플랫폼이 직접 입점 요청 시 우선 노출 가능

## 3. 차별점

### vs 아마존/쇼피파이
| 항목 | 아마존/쿠팡 | 새 플랫폼 |
|------|-----------|----------|
| 수수료 | 판매 수수료 5~15% | 없음 (어필리에이트 링크 그대로) |
| 대상 | 직접 판매자 | 어필리에이터 (누구나) |
| 범위 | 단일 국가 중심 | 전 세계 크로스보더 |
| 비용 표시 | 상품가만 | Total Landed Cost (관세+세금+배송 포함) |

### vs 기존 어필리에이트 마케팅
| 항목 | 블로그/유튜브 | 새 플랫폼 |
|------|------------|----------|
| 필요 역량 | 콘텐츠 제작, SEO, 영상 편집 | 상품 올리기만 |
| 도달 범위 | 본인 채널 팔로워 | 전 세계 (국가별 자동 노출) |
| 크로스보더 | 거의 불가능 | 핵심 기능 |
| AI 연동 | 없음 | AI 에이전트가 상품 데이터 활용 |

### 핵심 차별점 요약
1. **수수료 0%** — 어필리에이트 링크 그대로, 플랫폼이 안 뗌
2. **크로스보더 자동화** — Landed Cost 자동 계산, 국가별 자동 노출
3. **AI-ready** — 구조화된 상품 데이터를 AI 에이전트가 가져감
4. **진입장벽 제로** — 블로그/영상 안 만들어도 상품만 올리면 됨

## 4. 수익 모델

### Phase 1 (초기)
- **POTAL B2B 사이드 광고** — 플랫폼 사이드바에 POTAL 관세 API 광고
- **수수료 없음** — 셀러 유치에 집중

### Phase 2 (트래픽 확보 후)
- **광고 수익** — 사이드바/배너 광고 문의 받기
- **어필리에이트 플랫폼 입점비** — 우선 노출, 가이드 상단 배치
- **프리미엄 셀러 페이지** — 커스텀 디자인, 분석 도구

### Phase 3 (스케일)
- **광고 플랫폼화** — 트래픽 기반 광고 마켓플레이스
- **데이터 라이선스** — 상품/가격/수요 데이터
- **인수 가치** — 글로벌 상품 DB + 트래픽 + AI 데이터

## 5. 성장 플라이휠

```
셀러가 상품 등록
  → 상품 페이지 자동 생성 (SEO)
  → 셀러가 자기 페이지 공유 (자발적 마케팅)
  → 소비자/AI 트래픽 증가
  → 셀러 수익 발생
  → 더 많은 셀러 유입
  → 더 많은 상품 → 더 많은 트래픽
  → 어필리에이트 플랫폼이 입점 요청
  → 상품 폭발적 증가
  → 광고주 유입
```

핵심: **셀러가 올릴수록 플랫폼이 커지고, 커질수록 셀러가 더 온다.**

## 6. 기존 POTAL B2C 자산 활용

### 그대로 쓸 수 있는 것
- CostEngine (Landed Cost 계산) — POTAL B2B 엔진
- 검색/비교 UI 컴포넌트 (ResultsGrid, ProductCard, FilterSidebar)
- Provider 아키텍처 (데이터 정규화, 파이프라인)
- AI 분석 에이전트 (관련성 판단, 사기 필터링)
- 국가별 관세/세금 데이터 (240개국)

### 새로 만들어야 하는 것
- 셀러 가입/로그인 시스템
- 개인 페이지 (셀러 대시보드, 상품 관리)
- 상품 등록 폼 (어필리에이트 링크 + 상품 정보 입력)
- 국가별 어필리에이트 프로그램 가이드 페이지
- 어필리에이트 플랫폼 입점 시스템
- 국가 기반 상품 필터링 (배송 가능 국가 매칭)
- 광고 사이드바 시스템
- 별도 도메인/브랜딩

## 7. 메시지 (마케팅 방향)

### 셀러(어필리에이터)를 향한 메시지
> "AI가 모든 셀러들의 영역에서 셀러들을 대체하려 하고 있습니다.
> 하지만 AI는 직접적으로 모든 상품을 비교할 수 있는 디테일이 없습니다.
>
> 우리가 이걸 제공합니다.
>
> 세상의 모든 셀러들아, 너희들의 수익화를 할 수 있는 페이지를 제공합니다.
> 여기에 올려서 AI에게 제품 판매 영역을 뺏기지 마세요.
> AI가 여기서 너희들의 링크를 가져가게 만들 수 있습니다.
>
> 여기 모든 상품을 올려라. 너네가 판매하는 또 하나의 페이지가 생기는 거다."

### 핵심 훅
- **"블로그 안 만들어도 돈 번다"** — 상품만 올리면 전 세계 노출
- **"수수료 0%"** — 어필리에이트 링크 그대로, 안 뗌
- **"AI한테 뺏기지 마라"** — AI 시대에 셀러의 무기
- **"크로스보더로 돈 벌어라"** — 10개국 어필리에이트 가입하면 전 세계 수익화
