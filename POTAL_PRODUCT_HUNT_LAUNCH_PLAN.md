### 2. Description (설명)

```
POTAL is an API that calculates the total landed cost of importing
any product to any country.

What you get:
→ HS Code classification (100% accuracy with 9-field input)
→ Import duties, VAT/GST, customs fees — all in one API call
→ 240 countries, 113M+ tariff records, 63 Free Trade Agreements
→ MCP server for AI agents (Claude, ChatGPT, Gemini)

Free tier: 200 API calls/month. No credit card required.

Built by a solo founder. Feedback welcome!
```

### 3. Maker's First Comment (가장 중요한 요소)

```
Hey Product Hunt! 👋

I'm Eun, a solo founder building POTAL.

The problem: if you sell products internationally, calculating
import costs is painful. Every country has different duty rates,
VAT rules, de minimis thresholds, and trade agreements.
Most sellers guess — and get it wrong.

Existing solutions cost $1,500+/month (Avalara) or $2/order (Zonos),
and still only cover ~190 countries.

So I built POTAL — an API that calculates the exact total landed cost
for 240 countries, using 113M+ tariff records.

What makes it different:
• 240 countries (not 190 or 200)
• 100% HS Code accuracy with structured input (not AI guessing)
• 63 FTAs auto-detected for duty savings
• MCP server for AI agents — works with Claude, ChatGPT, Gemini
• Free tier: 200 calls/month, no credit card

The entire tariff engine is built on codified government rules —
not machine learning predictions. When you input 9 product fields,
the system follows the exact same process a licensed customs broker
would use.

What I'd love feedback on:
1. Is the API response format useful for your use case?
2. Which countries/products should I prioritize for testing?
3. Would you integrate this into your ecommerce platform?

Try it free → https://www.potal.app

Thanks for checking it out! Happy to answer any questions. 🙏
```

**포인트:**
- 자기소개 (solo founder = 커뮤니티 호감)
- 문제 정의 (공감대)
- 경쟁사 가격 비교 (차별점)
- 기술적 차별점 (codified rules, not AI guessing)
- 구체적 피드백 요청 (참여 유도)

## 댓글 대응 템플릿

**"어떤 이커머스 플랫폼 지원해?"**
```
Currently we have a Shopify app (in review), WooCommerce plugin,
and our REST API works with any platform.
MCP server also available for AI agent integration.
What platform are you using?
```

**"정확도가 어떻게 100%야?"**
```
Great question! The 100% accuracy applies when you provide 9 structured
fields (product name, material, category, etc).

We don't use AI to "guess" HS codes — instead, we codified 592 WCO rules,
1,233 headings, and 5,621 subheadings into deterministic code.
Same process a licensed customs broker follows, but automated.

With just a product name, accuracy drops significantly —
that's why we designed the API to accept structured input.
```

**"경쟁사 대비 장점이 뭐야?"**
```
Main differences:
• 240 countries (vs Avalara 190+, Zonos 200+)
• Free tier: 200 calls/month (vs $1,500+/month)
• 100% HS Code accuracy with structured input
• MCP server for AI agents (unique in this space)
• 113M+ tariff records in our database

We're an API-first infrastructure play —
we want to be the engine that powers other platforms,
not replace them.
```

**"무료인 이유가 뭐야?"**
```
Free tier (200 calls/month) is our way to let you evaluate
the API properly before committing.

Most businesses doing real cross-border commerce need 2,000+
calls/month — that's our Basic plan at $20/month.

We want to grow with our customers, not gate features behind paywalls.
```
