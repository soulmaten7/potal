## 너의 역할

너는 B2B SaaS 제품의 숏폼 콘텐츠 기획자야. 내가 만든 POTAL이라는 제품의 TikTok / Instagram Reels / YouTube Shorts 콘텐츠를 기획해줘.

## POTAL이 뭔지

POTAL은 크로스보더 이커머스(국가 간 온라인 쇼핑)를 위한 Total Landed Cost API야.
쉽게 말하면, 해외로 물건을 팔거나 살 때 "관세 + 세금 + 수수료"를 정확하게 계산해주는 서비스.

핵심 기능:
- HS Code 자동 분류: 상품명 입력하면 3초 만에 관세 코드 자동 분류 (100% 정확도)
- Landed Cost 계산: 관세 + VAT/GST + 수수료를 한 번에 계산 (240개국 지원)
- FTA 자동 적용: 자유무역협정 혜택 자동 적용해서 관세 절감
- 제재 심사 (Sanctions Screening): 거래 상대방이 제재 대상인지 확인
- 배치 분류: 수백 개 상품을 한 번에 분류

타겟 유저:
- 해외 판매하는 이커머스 셀러 (Shopify, WooCommerce 등)
- 물류/배송 회사
- AI 쇼핑 에이전트 개발자
- 크로스보더 결제 회사

경쟁사 대비 차별점:
- Zonos: 월 $4,000+ → POTAL: 월 $20부터
- Avalara: 월 $1,500+ → POTAL: 75배 저렴
- 240개국 커버리지 (경쟁사 최대 235개국)
- API 1번 호출로 전부 계산 (경쟁사는 여러 번 호출 필요)

홈페이지: https://potal.app
Product Hunt: https://www.producthunt.com/posts/potal

## 콘텐츠 요구사항

1. 숏폼 스크립트 5개를 만들어줘
2. 각 스크립트는 15~30초 분량
3. 화면 녹화 기반 (얼굴 안 나옴, 목소리 없음, 텍스트 자막 + 배경음악만)
4. 자막은 영어로 (글로벌 타겟)
5. 각 영상마다 "충격적인 사실" 또는 "문제→해결" 구조로 후킹

## 각 스크립트에 포함할 것

- 영상 제목 (TikTok/Reels 캡션)
- 해시태그 10개 (글로벌 크로스보더 관련)
- 초 단위 장면 구성 (0~3초: 후킹, 4~15초: 데모, 16~25초: 결과, 26~30초: CTA)
- 화면에 보여줄 텍스트 자막 전체
- 배경음악 분위기 추천

## 촬영 가이드도 만들어줘

- Mac에서 화면 녹화하는 방법
- 어떤 화면을 어떤 순서로 녹화할지
- 자막 넣는 무료 앱 추천
- 3개 플랫폼(TikTok/Reels/Shorts)에 동시 업로드하는 방법

## 영상 아이디어 힌트

이런 방향으로 만들어줘:
1. "HS Code를 아직도 수동으로 분류하세요?" → POTAL에서 3초 만에 자동 분류 데모
2. "일본으로 배송하면 진짜 비용이 얼마인지 아세요?" → Landed Cost 계산 데모
3. "당신의 거래처가 제재 대상일 수도 있습니다" → Sanctions Screening 데모
4. "$4,000/월 vs $20/월 — 같은 기능, 75배 가격 차이" → 경쟁사 가격 비교
5. "240개국 관세를 API 한 줄로" → 개발자 타겟 기술 데모

## 중요

- 과장 표현 하지 마. 정확한 사실만 사용해
- "100% HS Code 정확도"는 사실이니까 써도 됨
- "240개국"도 사실
- "$20/월 Basic 플랜"도 사실
- 경쟁사 이름(Zonos, Avalara)을 직접 언급하지 말고, "traditional solutions" 또는 "legacy providers" 같은 표현 사용
