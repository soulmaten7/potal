# Content — Marketing & Video Assets

This folder contains all marketing, video, and social media content assets for POTAL.

## Folder Structure

| Folder | Purpose |
|--------|---------|
| `demo-scripts/` | Feature demo video scripts (Excel). Each sheet = one dashboard menu. Follow the steps to record. |
| `social-media/` | Platform-specific post templates, hashtags, captions (LinkedIn, Twitter/X, Instagram, TikTok) |
| `thumbnails/` | Thumbnail images, banner sources, design files |
| `recordings/` | Finished video recordings (MP4, exported from Vrew/CapCut) |

## How to Use

1. Open `demo-scripts/POTAL_Dashboard_Demo_Scripts.xlsx`
2. Pick the sheet for the feature you want to demo
3. Follow Step → Action → Screen columns while screen-recording `potal.app/dashboard`
4. Import recording into Vrew, apply subtitles from columns D (Korean) and E (English)
5. Export and save to `recordings/`
6. Use `social-media/` templates for post captions when uploading

## Video Specs
- Resolution: 1920x1080 (YouTube) / 1080x1920 (Reels/TikTok)
- Length target: 30-90 seconds per feature
- Subtitles: Korean + English dual track
